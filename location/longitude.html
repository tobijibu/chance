longitude()

longitude()

// 使い方
chance.longitude()
chance.longitude({fixed: 7})

経度をランダムで返します。
範囲：-180 ～ 180

chance.longitude();
=> 149.41549

デフォルトでは、10進数の後に最大5桁の小数を返します。

fixedを指定すると、小数点以下の桁数を指定することができます。

chance.longitude({fixed: 7});
=> 51.4549925

デフォルトでは、経度の全範囲が含まれます。

min/maxを指定すると、最小経度/最大経度を指定することができます。

chance.longitude({min: -78, max: -77});
=> -77.22644

デモ

ボタンを押すと、経度をランダムに表示します。

結果：

JavaScript

var button = document.getElementById('get-result');
button.addEventListener('click', function() {
  var chance = new Chance();
  document.getElementById('result').innerHTML = chance.longitude({fixed: 7});
}, false);
