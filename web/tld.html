更新日: 2018.09.04

tld()

// 使い方
chance.tld()

デモ

ボタンを押すと、TLDをランダムで表示します。

結果：
JavaScript
var button = document.getElementById('get-result');
button.addEventListener('click', function() {
  var chance = new Chance();
  document.getElementById('result').innerHTML = chance.tld();
}, false);
