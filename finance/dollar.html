dollar()

更新日: 2018.09.04

dollar()

// 使い方
chance.dollar()
chance.dollar({max: 250})

ランダムなドル金額を返します。

chance.dollar();
=> "$2560.27"

chance.dollar();
=> "$750.99"

デフォルトでは、10000ドル以下の金額を返します。


maxを指定すると、最大金額を指定することができます。

chance.dollar({max: 20});
=> "$15.23"

chance.dollar({max: 10000000})
=> "$5051205.49"

デモ

ボタンを押すと、ドル金額をランダムに表示します。

結果：
JavaScript
var button = document.getElementById('get-result');
button.addEventListener('click', function() {
  var chance = new Chance();
  document.getElementById('result').innerHTML = chance.dollar();
}, false);
HTML
<button id="get-result">結果表示</button>
<div>
  <span>結果：</span>
  <span id="result"></span>
</div>
